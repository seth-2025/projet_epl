1)Qu'es ce que classes CSS

les classes permettent de définir plusieurs classes de contstes dans lesquels les noms en fomres sont différentes (voir partie 2 )

2) A quoi ça sert ?

Ainsi, des éléments qui ont le meme niveau d'imbrication dans les balises pourront avoir un style différent selon leur usage.

3)Exemple d'usage

Par Exemple, pour les liens 

  1. les liens dans un menu pourront avoir le style d'un boutton,
  2. alors que les liens dons article auront un aspect bleu souligné,
  3. ou encore les liens vers les sponsors dans le pied de page auront encore un autre aspect.
